A complete, fully functional e-commerce platform for sneaker sales built from the ground up. Features user authentication, product management, shopping cart, checkout workflow, seller dashboard, admin panel, and responsive design.

**Live Website:** [https://allaround2.netlify.app/](https://allaround2.netlify.app/)

## ✨ Core Features

### 🔐 Authentication & Security
- User registration with role selection (Customer/Seller)
- Secure login/logout functionality
- Role-based access control (Customer, Seller, Admin)
- Remember me option for persistent login

### 🛍️ Shopping Experience
- Browse sneakers by category (All, Lifestyle, Basketball, Casual, Skate)
- Product search functionality
- Add to cart with quantity management
- Real-time cart update and price calculation
- Shipping cost calculation based on order total
- Tax calculation (15% VAT)

### 👨‍💼 Seller Features
- Product listing with image upload
- Product management (Edit/Delete listings)
- Status tracking (Pending/Approved)

### 👑 Admin Features
- User management dashboard
- Product approval workflow
- Platform statistics overview
- Sales analytics and revenue tracking

### 📱 Responsive Design
- Mobile-first approach
- Works on desktop, tablet, and mobile devices
